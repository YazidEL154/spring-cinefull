# Exemple de projet avec spring

## Présentation

Le but de ce projet est de créer une API pour l'édition de Ticket de cinéma.

## Liste des besoins
* En tant que client je veux pouvoir :
    * X avoir accès a la liste des films
    * X avoir accès à la liste des seances
    * X connaitre le prix d'une seance
    * avoir accès a la liste des places assises.
    * X générer un ticket pour une seance
    * avoir la liste des seance pour un certain film
* Un ticket doit contenir les informations :
    * numero de seance
    * la salle
    * le titre du film
    * la place
    * date et heure de la seance
    * le cinema qui projette la seance
* En tant qu'administrateur de cinema je veux pouvoir:
    * ajouter un film
    * suprimer un film
    * ajouter une seance
    * supprimer une seance
    * definrir un prix a une seance
    * avoir le taux de remplissage de chaque seance
